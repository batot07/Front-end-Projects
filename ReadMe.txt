# Personal Portfolio Website (HTML Only)

## üìå Project Overview
This is a multi-page personal portfolio website built entirely using **semantic HTML5**. 

The goal of this project is to demonstrate a strong understanding of web structure, page linking, and SEO fundamentals without relying on CSS for styling. It serves as the foundational skeleton for a professional portfolio, featuring sections for projects, articles (resume/skills), and contact information.

## üöÄ Features
* **Multi-Page Navigation:** Includes four distinct pages linked via a consistent navigation bar.
* **Semantic HTML:** Utilizes tags like `<header>`, `<nav>`, `<main>`, `<footer>`, `<article>`, and `<details>` to ensure the code is meaningful and accessible.
* **SEO Optimization:** Each page includes `<meta>` tags for description, keywords, author, and Open Graph (OG) protocols for social media sharing.
* **Project Showcase:** A dedicated section to display projects with descriptions.
* **Resume/CV Content:** detailed "Articles" page listing skills, experience, certifications, and education.

4.  **`Contact.html`**
    * Provides multiple ways to get in touch.
    * Includes links to Email, LinkedIn, GitHub, and Behance.
